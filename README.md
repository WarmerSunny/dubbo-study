# dubbo 学习案例
## 项目概述：

​	dubbo版本  ：2.7.7

​	springboot版本：2.3.0.RELEASE

​	zookeeper版本：3.4.3

## 项目结构

- dubbo-study

  - common-api           公共接口包

  - dubbo-consumer   服务消费方

  - dubbo-provider      服务提供方
